# Currency Exchange Platform

A full-stack currency exchange platform that allows users to exchange currencies, manage transactions, and monitor rates through a modern and responsive interface.

## Features

### User Features
- Currency conversion
- Real-time exchange rate updates
- Real-time exchange rate conversion
- Real-time notifications
- Analytics dashboard
- Transaction management
- Responsive UI
- Secure authentication
- Google OAuth login

### Admin Features
- Manage exchange rates
- Add/update payment methods
- Monitor transactions
- Pagination and filtering
- Admin dashboard

# Installation

## Clone the repository

```bash
git clone https://github.com/yourusername/currency-exchange-platform.git
```

## Install frontend dependencies

```bash
cd client
npm install
```

## Install backend dependencies

```bash
cd server
npm install
```

---

# Running the Application

## Start Backend

```bash
npm run dev
```

## Start Frontend

```bash
npm run dev
```
